# Contributing ドキュメント

このフォルダには、MissionAIプロジェクトへのコントリビューションに関するドキュメントが含まれています。

## 📁 ファイル構成

### コントリビューションガイド

#### `CONTRIBUTING.md` ⭐ **最重要**
- **ステータス**: アクティブ（コントリビューションガイド）
- **用途**: コントリビューション方法、コーディング規約、プルリクエストの手順
- **対象読者**: コントリビューター
- **内容**:
  - コントリビューションの種類
  - 開発環境のセットアップ
  - コーディング規約（TypeScript/JavaScript、Rust）
  - ブランチ戦略
  - コミットメッセージ
  - プルリクエストの手順
  - コードレビュー
  - テスト
  - ドキュメント
  - 質問とサポート
  - 行動規範

## 🔗 ドキュメント間の関係性

```
CONTRIBUTING.md (コントリビューションガイド)
    ├─→ ../setup/SETUP_GUIDE.md（開発環境セットアップの参照）
    ├─→ ../development/DEVELOPMENT_GUIDELINES.md（開発ガイドライン）
    └─→ ../troubleshooting/TROUBLESHOOTING.md（トラブルシューティング）
```

## 📖 読み方のガイド

### 新規コントリビューター

1. **まず読む**: `CONTRIBUTING.md`（コントリビューション方法の確認）
2. **次に読む**: `../setup/SETUP_GUIDE.md`（開発環境のセットアップ）
3. **参考**: `../development/DEVELOPMENT_GUIDELINES.md`（開発ガイドライン）

### 既存コントリビューター

1. **参考**: `CONTRIBUTING.md`（コントリビューション手順の確認）

## 📝 ドキュメントの更新方針

- **コントリビューションガイド** (`CONTRIBUTING.md`): コントリビューション手順の変更時に更新
- コーディング規約の変更時に更新
- ブランチ戦略の変更時に更新
