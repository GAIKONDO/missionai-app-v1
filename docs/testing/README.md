## 📖 読み方ガイド

### テストを実行する場合

1. **TESTING_DESIGN_DOC_SECTIONS.md** を読む

3. **次のステップ** を確認
   - 今後の改善計画を理解

## 📌 注意事項

1. **ChromaDBの起動**: 
   - 埋め込み生成機能を使用する場合、ChromaDB Serverが起動している必要があります
   - ChromaDBが起動していない場合、セクション作成自体は成功しますが、埋め込み生成は失敗します

2. **テーブル名**: 
   - SQLite: `designDocSections`テーブル
   - ChromaDB: `design_docs`コレクション（組織ごとではなく、全体で1つのコレクション）

3. **埋め込み生成**: 
   - セクション作成時に自動的に埋め込みが生成される場合があります
   - 手動で埋め込みを生成する場合は、システム設計ページの「インデックス」ボタンまたは「埋め込み生成」ボタンを使用

4. **テスト環境**: 
   - 開発環境でのテストを想定しています
   - 本番環境でのテストは、適切なバックアップを取ってから実行してください
